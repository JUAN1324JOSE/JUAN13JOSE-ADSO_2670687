Concepto de GIT

GIT es un software de control de versiones o que significa que un clon local del proyecto es un repositorio de control de versiones completo. Estos repositorios locales plenamente funcionales permiten trabajar sin conexión o de forma remota con facilidad.

LOS PRINCIPALES COMANDOS DE GIT

  • GIT INIT: Este comando funciona para iniciar un proyecto con git
  • GIT ADD: Permite agregar todos los documentosal
  • GIT CONFIG: Es una función útil que sirve para definir valores de configuración de Git a nivel de un proyecto global o local
  • GIT STATUS: Muestra el estado del directorio de trabajo y del área del entorno de ensayo
  • GIT COMMIT: Captura una instantánea de los cambios preparados en ese momento del proyecto.
  • GIT BRANCH: Te permite crear, enumerar y eliminar ramas, así como cambiar su nombre. No te permite cambiar entre ramas o volver a unir un historial bifurcado.
  • GIT CHECKOUT: Te permite desplazarte entre las ramas creadas por git branch
